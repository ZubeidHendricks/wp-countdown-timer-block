Drop a live countdown timer anywhere with a shortcode — launches, sales, events. Lightweight, no jQuery.

== Description ==

Add urgency to launches and sales with a clean, live-updating countdown. Use the
`[countdown date="2026-12-31 23:59"]` shortcode anywhere — posts, pages, or page
builders. When it hits zero it shows your custom expiry message.

**Free features**

* Days / hours / minutes / seconds, updating live.
* Custom accent colour and expiry message.
* Works in posts, pages and most builders.
* No jQuery, tiny footprint.

**Pro features**

* Evergreen, per-visitor countdowns.
* Auto show/hide content on expiry.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Add `[countdown date="2026-12-31 23:59"]` to any content.

== Frequently Asked Questions ==

= What date format does it accept? =

Anything PHP's strtotime understands, e.g. `2026-12-31 23:59` or `December 31 2026`.
